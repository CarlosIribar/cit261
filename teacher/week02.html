Instructor Preparation

Overview

The first team activity happens this week. There will be one most of the weeks. You can either designate groups, or let students self-select them. The activities are intended to be done in peer-programming style. Because this course is online, you might suggest something like multiplayer in repl.it to code together. Screen sharing in Zoom or Hangouts can work as well. The teams need to schedule an hour at some point during the week for the team activity. This week they will build a simple adding machine.

Encourage them also to look to other sources when things don't make sense. The book does a good job, but the way it addresses certain topics might be confusing to some students. Help them remember that there are other options. Several are listed in this weeks reading. The notes they turn in do not have to be all from the assigned reading.

Encourage the students to record their questions while reading and give them to you to be addressed.

Meeting Time

Since the reading introduced object literals, mention in your meeting that almost everything in JavaScript (JS) is an object with properties and methods. Review the three ways to declare functions (declaration, expression, arrow) and mention that since functions are just objects, they can be bound to variables, passed as arguments in functions (callback) or made part of another object.

You can use something like console.dir() on an array to show them what an array object looks like as well to introduce the array methods. (array.includes(element) .push .pop .shift .unshift .slice .split .reduce .filter array.map(callback)). Keep in mind that the team activity for next week will also be on the array methods.

Your demo should show examples of using const and let, and contain a review of the three methods for declaring functions in JS (function expression, function declaration, and arrow function). You could build on the calculator code from the team activity (partially converted using an object or fully converted to an object), or build a Random Sentence Generator Example.
